Relations & Functions

Relation: a set of ordered pairs
Elements: Members of a set
Domain: Elements from the 1st set
Range: Elements from the 2nd set
Functions: A relation in which each element in the domain is paired with exactly one element in the range
Vertical Line Test: Slicing through a graph with an imaginary line where if any x value has 2 or more y-values then the graph is not a function


Function Notation
$y=f(x)$
$y=f(x)={x^2}/{x+1}$
$f(-2)={(-2)^2}/{-2+1}={4}/{-1}=-4$

Example
