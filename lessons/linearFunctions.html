Linear Functions
$f(x)=mx+b$, where $m,b ∈ R$
Zeros of a Function - values of $x$ for which $f(x)=0$
Ex) $f(x)=1/2x-2$
$\table 0=1/2x-2; 2=1/2x; 4=x$
