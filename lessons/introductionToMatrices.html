Equal Matrices
Same dimensions $(\table m, x, n)$
Correspoding elements are equal

Ex) Find the values of x & y for which the following equation is true
$[\table y-2;2x]=[\table 2x-8; 4y]$

$\table y-2=2x-8; 2x=4y$

$\table 2x=4y; x=2y$

$\table y-2=2x-8; y-2=2(2y)-8; 6=3y; 2=y$

$\table x=2(2); x=4$

Adding & Subtracting Matrices
Must be same dimensions $(\table m, x, n)$
Add or subtract the corresponding elements

Ex) Let $A=[\table 2, -4; 3, -8]$ & $B=[\table -5, 7; 1, 4]$
Find $A+B=[\table 2+(-5), -4+7; 3+1, -8-4]=[\table -3, 3; 4, -4]$
$A-B=[\table 2-(-5), -4-7; 3-1, -8-4]=[\table 7, -11; 2, -12]$

Scalar Product
Multiply each element of the matrix by the scalar, $k$
Ex) Given $k=3$ & $A=[\table 2, 5, -1; 3, -2, 4]$, find $kA$
$\table kA=3[\table 2, 5, -1; 3, -2, 4],=[\table 3(2), 3(5), 3(-1); 3(3), 3(-2), 3(4)]; ,=[\table 6, 15, -3; 9, -6, 12]$
