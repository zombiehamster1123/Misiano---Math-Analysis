Cramer's Rule
Used to solve for variables
$\table \table x, =, {D_x}/D, \table y, =, {D_y}/D, \table z, =, {D_z}/D$
System of Equations Determinant of Coefficient Matrix Answer Column Coefficient Determinant with answer-column values in x-column
$\table 2x+1y+1z=3; 1x-1y-1z=0; 1x+2y+1z=0$ $D={|\table 2, 1, 1; 1, -1, -1; 1, 2, 1|}$ $[\table 3; 0; 0]$ $D_x=|\table {\html'<span style="color:red">3</span>'}, 1, 1; \html'<span style="color:red">0</span>', -1, -1; \html'<span style="color:red">0</span>', 2, 1|$

Similarly, $D_y$ and $D_z$ would then be:
$\table D_y=|\table 2, \html'<span style="color:red">3</span>', 1; 1, \html'<span style="color:red">0</span>', -1; 1, \html'<span style="color:red">0</span>', 1| D_z=|\table 2, 1, \html'<span style="color:red">3</span>'; 1, -1, \html'<span style="color:red">0</span>'; 1, 2, \html'<span style="color:red">0</span>'|$

Evaluating each determinant, you get...
$\table D=,{|\table 2,1,1; 1,-1,-1; 1,2,1|} =2{|\table -1,-1; 2,1|}-1{|\table 1,-1; 1,1|}+1{|\table 1,-1;1,2|};,=2(1)-1(2)+1(3);,=3$
$\table D_x=,{|\table 3,1,1; 0,-1,-1; 0,2,1|}=3{|\table -1,-1; 2,1|}-1{|\table 0,-1; 0,1|}+1{|\table 0,-1; 0,2|};,=3(1)-1(0)+1(0);,=3$

$\table D_y=,{|\table 2,3,1; 1,0,-1; 1,0,1|}=2{|\table 0,-1; 0,1|}-3{|\table 1,-1; 1,1|}+1{|\table 1,0; 1,0|};, =2(0)-3(2)+1(0);,=-6$

$\table D_z=,{|\table 2,1,3; 1,-1,0; 1,2,0|}=2{|\table-1,0; 2,0|}-1{|\table 1,0; 1,0|}+3{|\table 1,-1; 1,2|};, =2(0)-1(0)+3(3);,=9$


$x={D_x}/D=3/3=1$
$y={D_y}/D=-6/3=-2$
$z={D_z}/D=9/3=3$

Ex) Given the following system, solve for z: $\table 2x+y+z=1; x-y+4z=0; x+2y-2z=3$

$1^{st}$, find coefficient determinant
$D= |\table 2,1,1; 1,-1,4; 1,2,-2|=2|\table -1,4; 2,-2|-1|\table 1,4; 1,-2|+1|\table 1,-1; 1,2|$
$D=2(-6)-1(-6)+1(3)=-3$

$2^{nd}$, find $D_z$
$D_z=|\table 2,1,1; 1,-1,0; 1,2,3|=2|\table -1,0; 2,3|-1|\table 1,0; 1,3|+1|\table 1,-1; 1,2|$
$D=2(-3)-1(3)+1(3)=-6$

$3^{rd}$, divide & simplify
$z=D_z/D={-6}/{-3}=2$   $z=2$

If the coefficient determinant=0, you can't use Cramer's Rule
